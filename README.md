# AWS Consulting Playbook 🚀

A complete framework for delivering AWS cloud consulting engagements — from first client conversation to final delivery.

Built by AWS Solutions Architect (SAA)

---

## What Is This?

This playbook captures everything needed to successfully deliver AWS cloud projects as a consultant. It includes templates, frameworks, code starters, and reference guides built from real project experience.

**The Fantasy Sports Helper project** — a 23-service AWS application built from scratch — served as the foundation for this playbook.

---

## Who Is This For?

- Cloud consultants delivering AWS projects
- Developers transitioning into cloud consulting
- Anyone building AWS applications who wants a structured approach

## The Consulting Framework

Every engagement follows this pattern:

| Phase | What Happens | Key Deliverable |
|-------|-------------|----------------|
| **Discover** | Understand the client's problem | Discovery questionnaire |
| **Analyze** | Document current state and pain points | Current state process map |
| **Design** | Plan the solution | Architecture diagram + requirements |
| **Build** | Create the solution on AWS | Working application |
| **Deliver** | Hand over and prove value | ROI report + handover doc |

## Getting Started

1. Start with the **Discovery Questionnaire** in `ba-templates/`
2. Use the **ROI Calculator** in `roi-calculator/` to build your business case
3. Pick an **Architecture Template** from `architecture-templates/`
4. Use **Code Templates** from `code-templates/` to jumpstart development
5. Deliver using templates in `client-deliverables/`
